This code is developed in MERN stack.The UI designs that was created to simulate the same look for the website are present in the My UI DESIGNS folder.
VS CODE is used as the code editor .
To start with a ReactJs application do the following steps:

npx create-react-app secure_cart //This will fetch basic requirements for the app and structure for the codebase
To install node modules , run npm install.
To run the application , please use npm start in vscode terminal by navigating to the project folder.

4 main pages are planned forthe UI of this application web application:
1) Login
2) shoppingcart
3) search users 
4) contact us

This directory consist of the Froentend codebase- components ,css, node_modules and other important UI rendering components.
